# <AREA_NAME> / <DOMAIN_NAME> Module

Этот модуль задаёт глубокую специализацию агента для домена `<DOMAIN_NAME>` в области `<AREA_NAME>`.

## Содержимое

- `rules/` — обязательные инварианты (всегда активны).
- `skills/` — подключаемые компетенции (по задаче).
- `workflows/` — пошаговые алгоритмы выполнения.
- `prompts/` — готовые шаблоны запросов и системный контекст.

## Принцип заполнения

1. Сначала зафиксируй инварианты в `rules/`.
2. Затем опиши повторяемые сценарии в `workflows/`.
3. После этого добавь skills для узких задач.
4. В конце собери prompts, которые ссылаются на rules/skills/workflows.
